# Enter-Price-Data-Warehouse-Reporting-System
The Enterprise Sales Reporting Dashboard is a Business Intelligence project designed to analyze and visualize sales performance using a centralized Data Warehouse. The project integrates data processing, storage, and reporting techniques to transform raw sales data into meaningful business insights.

The sales data is collected, cleaned, and transformed using Python and Pandas, then stored and analyzed using SQL. A Data Warehouse is created to organize the data efficiently for reporting purposes. Finally, Power BI is used to build an interactive dashboard that provides visual insights into key business metrics such as Total Sales, Profit, Quantity Sold, and Order Count.

The dashboard includes various visualizations such as Sales by Year, Sales by Category, Sales by Segment, Profit by Region, Sales by State, and Top-Selling Products. These reports help businesses identify trends, monitor performance, and make data-driven decisions.

This project demonstrates practical skills in Data Warehousing, ETL Processes, SQL Querying, Data Analysis, Data Visualization, and Business Intelligence Reporting, making it suitable for real-world business analytics applications.
